# 代码片段

一系列代码片段，用于快速创建带有相关功能和检查的进一步模板代码。

我从常规使用和反复编写的代码中选择了这些代码片段。与其再次包含导致很多困惑的 `__templates/` 目录，不如将其添加为工作区代码片段库。这也避免了创建和维护扩展的问题。

## 代码片段

| 代码片段                 | 描述                                |
| ------------------------ | ----------------------------------- |
| `sfc`                    | 创建箭头函数组件                    |
| `createNewPage`          | 创建带有 Card 的新 Polaris 页面     |
| `createapi`              | 为 `/api` 创建新端点                |
| `createproxy`            | 为 `/api/proxy_route` 创建新端点    |
| `createcheckout`         | 为 `/api/checkout` 创建新端点       |
| `createwebhook`          | 创建新的 webhook 函数               |
| `createOnlineClientGql`  | 使用在线令牌创建新的 GraphQL 客户端 |
| `createOfflineClientGql` | 使用离线令牌创建新的 GraphQL 客户端 |
| `createpost`             | 创建新的 POST 请求                  |
| `createget`              | 创建新的 GET 请求                   |
